# Tools

Shared tools and scripts to support development.

## Scripts

Utility scripts:

- `create-project.sh`: Create a new project from a template
- `setup-env.sh`: Setup development environment
- `deploy.sh`: Deploy script
- `clean.sh`: Clean build artifacts

## Generators

Code generators for creating boilerplate:

- Component generators
- API route generators
- Test file generators

## Usage

```bash
# Make scripts executable
chmod +x tools/scripts/*.sh

# Run a script
./tools/scripts/create-project.sh my-project react
```
