# Persona: First-time DeFi User

## Profile

- **Name:** Alex
- **Age:** 25
- **Background:** Has used Binance CEX, transitioning to DeFi for the first time
- **Crypto experience:** Knows how to buy/sell on CEX, never interacted with a smart contract
- **Wallet:** Just set up MetaMask last week, holds ~$200 ETH

## Mental Model

- Thinks DeFi is like Binance but without sign-up
- Does not understand what gas fees are or why extra fees are required
- Cannot distinguish between token approve and swap
- Does not know slippage, impermanent loss, or liquidity pools
- Expects everything to be instant like CEX

## Behaviors

- Skims text, does not read long paragraphs
- Clicks the biggest, most prominent button on the page
- Panics when seeing red warnings, unsure whether to proceed
- Does not know to verify contract addresses
- Copy-pastes wallet addresses from friends without verifying

## Pain Points

- DeFi terminology: slippage, liquidity, TVL, APY vs APR, impermanent loss
- Cannot distinguish mainnet vs testnet
- Does not understand why approval is needed before a swap
- Afraid of losing funds, does not know if transactions are reversible
- Confused when multiple tokens share the same name across different chains

## Key Questions (always ask during review)

1. "What does this mean?" — Is there a tooltip or explanation?
2. "Will clicking this cost me money?" — Is risk clearly communicated?
3. "What do I do next?" — Is there post-action guidance?
4. "Why is this taking so long?" — Are loading/pending states clear?
5. "Something went wrong, how do I fix it?" — Are error messages actionable?
