# Persona: Experienced DeFi User (Degen)

## Profile

- **Name:** Minh
- **Age:** 30
- **Background:** Trading DeFi since DeFi Summer 2020, uses 10+ protocols
- **Crypto experience:** Understands smart contracts, reads Etherscan, knows MEV
- **Wallet:** Multi-wallet setup, hardware wallet for large amounts

## Mental Model

- Knows the full flow: connect wallet → approve → execute → confirm
- Understands gas optimization, knows when gas is cheapest
- Compares rates across multiple DEXs before swapping
- Cares about security: contract verified, audit report, TVL trends
- Expects pro-level features: limit orders, advanced routing, analytics

## Behaviors

- Quickly scans the page, looks for key metrics first
- Checks contract address on explorer before approving
- Sets custom slippage and gas price
- Uses keyboard shortcuts when available
- Opens multiple tabs to compare protocols

## Pain Points

- UI too simplistic, lacks advanced options
- Does not show routing path — unclear where the swap is routed
- Missing price impact warning for large trades
- No detailed transaction history on the UI
- Rate refresh too slow, quote outdated at execution time

## Key Questions (always ask during review)

1. "Is this rate good?" — Is there a comparison or routing breakdown?
2. "Is this contract safe?" — Is there an audit link or verified badge?
3. "How much am I actually paying in fees?" — Is fee breakdown clear?
4. "Is there an advanced mode?" — Are pro features accessible?
5. "Where is my history?" — Does transaction tracking have enough detail?
6. "Why did it fail?" — Does the error show the revert reason?
